Core Analytical Capabilities

  • Advanced Segmentation: Adaptive thresholding & watershed for high-porosity samples.
  • Network Extraction: "Ball-and-Stick" models compatible with OpenPNM.
  • Clipping Analysis: Clips dataset by ROI, threshold, or mask.
  • Performance: Handles >1GB datasets via chunked GPU processing within seconds.

Algorithm Details

Watershed Separation

Splits touching pores via Distance Transform

Network Extraction

Abstracts geometry to Nodes & Links

  • Adaptive Binarization: Auto-selects Otsu (bimodal) or Triangle (unimodal) thresholding.
  • Seeded Watershed: Uses local maxima of Distance Transform as markers to prevent over-segmentation.
  • Max-Ball Algorithm: Defines pore/throat sizes by the largest inscribed sphere at centroids and boundaries.

Future: Time-Resolved (4D) CT

Goal: Monitor dynamic processes (Baking, Frying, Compression) in real-time.

Applications

  • Bubble nucleation rates during proving.
  • Water wavefront propagation (Rehydration).
  • Structural collapse under compression.

Technical Challenges

  • Data Volume: 4D datasets explode in size (TB scale), requiring streaming processing.
  • Topological Tracking: Extremely difficult to map "Ball-and-Stick" models across timeframes as pores merge/split.
